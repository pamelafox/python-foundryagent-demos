# Summary

You've completed the lab on building agentic knowledge bases with Azure AI Search and MCP integration. Through 5 progressive notebooks, you explored how to build multi-source knowledge bases, expose them as MCP endpoints, connect live MCP servers as knowledge sources, and wire everything into a Foundry Agent.

## What you explored

In this lab, you built a knowledge base with indexed enterprise content (HR and health documents), exposed it as an MCP endpoint for developer tools, connected non-authenticated (Microsoft Learn) and authenticated (GitHub) MCP servers as live knowledge sources, and created a Foundry Agent grounded on your knowledge base for conversational AI experiences.
